
# IndianCab — React + Vite + TypeScript (Lenis smooth scroll)

A light theme landing + pages site for IndianCab with gold accents, WhatsApp booking, and dedicated pages for Services, Fleet, Features, USP, and Contact. Image areas are replaceable `.img-placeholder` divs.

## Quickstart
```bash
npm i
npm run dev
```

## Replace Logos & Images
- Replace `src/assets/logo.png` (provided) and/or `logo-white.svg` if needed.
- Replace `.img-placeholder` with `<img/>` or real components.
